The Author

Hello! I'm Victor Ferreira Araújo, and I developed this project as a student at Conestoga College, Canada. I was born in Brazil and worked as a Software Engineer for a couple of years before becoming a student again.

Now, I'm living in Kitchener, Ontario, and I want to sum up my previous professional experiences with my academic ones to do my best in the Canadian IT market.

Program and Course

The program I'm taking is Web Development ( PROG8661 ), and the course related to this project is JavaScript Winter 2025 Section 2, administrated by Professor John Prinz.

The Project

Professor John requested this project as an assignment, and the goal is to use the course's content to build any web app, in this case, a word-guess/picture-reveal game using HTML5, CSS3, and JavaScript.

For this project, I used the jQuery library at version 3.7.1-slim to handle events, elements selection, DOM manipulation and so on. This was an amazing project for me to get a hands-ok practice with the Web's most important triplet ( HTML, CSS and JavaScript ).
